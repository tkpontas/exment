Exment Ver.6系をベースに公式では取り込まれなかった機能を勝手に追加したバージョンです。
機能の開発は私がしたものではありません。なお、公式リリースされていないため、バグ不具合があっても、公式issueに報告しないでください。
（おそらく報告しても対応してもらえないと思います）

<公式で取り込まれなかった機能（追加した機能）>
- feature/issue_9002 （検索フィルタで条件を保持したままにする）
- feature/add_expansion_view （1:Nリレーションのテーブルに親子ビューの追加）※ lang/enに不足項目追加済
- feature/issue_606_ex（パラメータ値を使用した表示専用で検索不可の項目をカスタム列に追加）
